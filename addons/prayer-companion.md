# 🕌 Prayer Companion — Addon

Prayer time reminders with Quranic verses for each salah.

**Commands:** `/prayer`, `/prayer settings`
**Requires:** User timezone, optional location
**Features:**
- Prayer time notifications
- Suggested verse for each salah
- Fajr: verses about morning/guidance
- Dhuhr: verses about work/provision
- Asr: verses about patience
- Maghrib: verses about gratitude
- Isha: verses about forgiveness/sleep
