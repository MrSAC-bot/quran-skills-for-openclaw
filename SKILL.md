# 📖 Quran Skills for OpenClaw

> 58 Quranic verses with Arabic recitation + English translation + your choice of 3rd language.
> 16 categories covering everyday life: Healing, Protection, Forgiveness, Patience, and more.

## What's Included

- **Arabic** — Mishary Al-Afasy recitation ✅ (always included)
- **English** — Yusuf Ali translation + Ryan TTS ✅ (always included)
- **Your Choice** — Pick 1 language at install time 🎯

## Language Selection

When you install, you choose your 3rd language:

```
🌍 Vælg dit sprog:

  [🇵🇰 Urdu]  Ahmed Ali
  [🇹🇷 Turkish]  
  [🇪🇸 Spanish]  
  [🇫🇷 French]   
  [📥 Download other language...]
```

Only the language you pick is downloaded — keeps the skill lightweight.

## Commands

| Command | Action |
|---------|--------|
| `/dua` | Show all 16 categories |
| `/dua healing` | Get Healing category with audio |
| `/search heart heavy` | Search verses by keyword/feeling |
| `/aya 2:255` | Get specific verse |
| `/surah` | Get a surah with audio |
| `/random` | Random verse |
| `/daily` | Verse of the day |
| `/help` | Show help |
